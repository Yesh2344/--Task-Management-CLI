# Task Management CLI

A simple command-line interface for managing tasks. This tool allows you to add, list, mark as complete, and delete tasks.  It stores tasks in a JSON file, ensuring persistence between sessions.

## Features

*   **Add Tasks:** Easily add new tasks with a description.
*   **List Tasks:** View all tasks, with status and ID.
*   **Mark as Complete:** Update the status of a task to "completed".
*   **Delete Tasks:** Remove tasks by their ID.
*   **Persistent Storage:** Tasks are stored in a JSON file, ensuring data is preserved.

## Installation

1.  Clone the repository:

    ```bash
    git clone <your_repository_url>
    cd task_management_cli
    ```

2.  Install dependencies:

    ```bash
    pip install -r requirements.txt
    ```

## Usage

Run the script with the `main.py` file.
